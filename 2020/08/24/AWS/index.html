AWS Day1

Amazon Web Services Introduction

クラウドとは︖

クラウドとは、⾃社でハードウェアや設備を購⼊、準備することなく、インターネット経由でコン ピューティング、データベース、ストレージ、ソフトウェアといったさまざまな IT リソースをオン デマンドで利⽤することができるサービスの総称 利⽤までにかかる時間の圧倒的短縮、需要の縮⼩/拡張にあわせたリソースの利⽤、利⽤した分だけ の⽀払いという従量課⾦制、IT 資産の固定費から変動費への転換といった特徴がある

Our Vision

地球上でもっともお客様を⼤切にする企業であること

  1. コンシューマー ビジネス

  3. IT インフラ ビジネス

AWS の誕⽣の背景

Amazon のビジネスの成⻑スピードを緩めない

  • システムの拡張性

  • 機能の充実

セルフサービスプラットフォームへの挑戦

  • 徹底した⾃動化
  • API で利⽤できる基盤
  • モノリシックからの脱却
  • マイクロサービス化
  • 管理者の介在を極限まで削減 徹底した⾃動化 IT

→ AWS は 2006 年から稼働 (⽇本では2011年から東京リージョンを稼働)

AWSが選ばれる主な6 つの理由

１．俊敏性

  • ・数千台のサーバを数分で提供
  • 175種類以上のサービス
  • すぐに始められ、すぐに⽌められる

２．コスト削減

  • お客様の体験を向上するための投資を⾏い、コストダウンを続けています。
  • TCO(Total Cost of Ownership)の低減

３．弾⼒性

  • 事業の必要性に応じて即座にスケールアップ、スケールダウンが可能です。

４．幅広いサービスと機能

  • 幅広いサービスと機能改善・イノベーションのスピードがあります。
  • 175を超えるサービス数
  • 技術への継続投資によるイノベーションのご提供

５．グローバルインフラストラクチャ

  • わずか数分でグローバル規模の展開を実現可能

災害対策および地理的な要件に対応
