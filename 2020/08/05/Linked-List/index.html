160. Intersection of Two Linked Lists

  1. 解题思路：

    双指针+链表拼接

    两个指针i，j同时一起走，

    如果i指向的元素不在headB里面（找共同的元素），则i.next，往前走

    如果j指向的元素不在headA里面，则j.next，往前走

    设 A 的长度为 a + c，B 的长度为 b + c，其中 c 为尾部公共部分长度，可知 a + c + b = b + c + a。
