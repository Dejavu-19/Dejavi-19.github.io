What Are Data Structures?

The elementary particles of algorithms, data structures are woven into the very fabric of computer science and are essential building blocks of many a solution to coding interview problems. Sans the need for a microscope.

Coding or software engineering is manipulating data

web app :

  1. working at frontend—making requests to—backend get data
  2. transforming data into human readable format
  3. display data on a screen maybe on a website or on a phone screen

developing an API :

  1. stores and retrieves data from a database that transforms that data a little bit
  2. returns it to a client that asks for it

running joke at big tech companies

All that software engineers do, on a day-to-day basis during their jobs, is move data from one place to another, from one service to another.

Data structures are really just a way to organize and manage data.

Wikipedia puts it pretty elegantly

A data structure is a collection of data values, the relationships among them, and the functions or operations that can be applied to the data.

Complexity Analysis

Consists of two dimensions, time and space, is sort of like the bedrock of coding interview problems.

Maybe you’ve even coded it out your interviewer’s going to ask you

the notorious question

can you do better?

Can you find a better solution?

So what makes one solution better than another solution?

And this is where complexity comes into play.

Granted, if a solution is needlessly complicated,that might make it worse than another solution,

time complexity and space complexity

  1. The faster an algorithm runs,the better its time complexity is and therefore the better the algorithm is.
  2. Similarly, the less memory an algorithm takes,the better its space complexity is and therefore the better the algorithm is.

Different data structures are gonna have different time complexity and space complexity ramifications for the functions and operations that they support.

not only to figure out what data structure best allows you to solve the problem,but also what data structure allows you to do so with the best time complex it you with the best space complexity.

This means that you’re gonna have to be very familiar with all of the space-time complexity ramificationsof the various popular data structures.
