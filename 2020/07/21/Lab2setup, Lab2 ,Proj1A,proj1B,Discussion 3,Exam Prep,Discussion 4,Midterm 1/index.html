Raining Cats and Dogs

a = new Dog(“Spot”, 10);

d = a;

Why would this code produce a compiler error? How could we fix this error?

A: The static type of d is Dog while the static type of a is Animal. Dog is a subclass of Animal, so this assignment will fail at compile time because not all Animals are Dogs. Use casting to address the problem.

d = (Dog) a;
