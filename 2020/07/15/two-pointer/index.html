167. Two Sum II - Input array is sorted (Easy)

Solution1 :

要求：you may not use the same element twice.

  1. two pointer O(n logn) O(1) good sorting algorithms like Quicksort or Merge sort or Heapsort O(n logn)
  2. Let the two pointers i and j correspond to the head and tail of the array, respectively, and sum the two. num[i]+num[j] = k
  3. If k is greater than traget, perform j-1 operation (because of ascending order, the tail number is large);
    If k is smaller than target, perform i+1 operation.

633. Sum of Square Numbers

Solution1 :

  1. For a non-zero integer c, is there a^2^+b^2^ = c
    Consider x^2^ = c, the value range of x is 0- \sqrt c
  2. Set the double pointer, set i to be in the head of the value range c i = 0, j to be in the value range j = sqrt c, automatically rounded
    If i2+j2> c, the right end j-1
    If i2+j2 <c, the left end i+1
    No integer makes the return Flase when the equation holds

345.Reverse Vowels of a String

Solution1 :

  1. Using two pointers, one pointer traverses from head to tail, and one pointer traverses from end to head. When both pointers traverse to vowel characters, the two vowel characters are swapped.

  2. In order to quickly determine whether a character is a vowel character, we add all the vowel characters to the set HashSet, so that the operation is performed with O(1) time complexity.

    Time complexity is O(N): only need to traverse all elements once
    Space complexity O(1): only two additional variables need to be used

    contains返回两个String是否包含。

    “abcdefg”.indexOf(“c”) // 返回2
