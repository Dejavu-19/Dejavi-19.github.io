  6. manchester: 这个有两处错误，是要判断array里每位数字和前面一位是不是相等，index0的数字单独和0判断一下，所以0那句不能直接res[0]= 0要res[0] = (arr[0] != 0) ? 1:0，后面for循环里判断是要把arr == arr[i-1]改成arr != arr[i-1]。曼切斯特如果arr[i-1],arr相等为0，否则为1，要注意不但==要改成!=，ret[0]也要加一下，不然有一个case过不了。

  14. remove Duplicates: for循环里k <length改成k < length-1

  15. sum Array: sum =arr改成sum += arr

  16. array Operation: <和>写反了111111111111111111 《=，去=， arr[j] -> 缩进

  17. print1111 : 外层加个括号，外层的for少大括号
